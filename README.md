## 🚀 Возможности <svg xmlns="http://www.w3.org/2000/svg" width="38" height="38" viewBox="0 0 38 38" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" style="vertical-align: middle; margin-left: 6px;"><path d="m5 8 6 6"/><path d="m4 14 6-6 2-3"/><path d="M2 5h12"/><path d="M7 2h1"/><path d="m22 22-5-10-5 10"/><path d="M14 18h6"/></svg>

* **Автоматический перевод** синхронных текстов песен с помощью Google Translate (бесплатный клиент gtx).
* Перевод отображается **под оригинальной строкой**, не перекрывая её и не мешая.
* **Горячие клавиши** для быстрого управления.
* **Настройка** внешнего вида перевода.

## ⌨️ Горячие клавиши

| Сочетание | Действие |
| :--- | :--- |
| `Ctrl + Shift + T` | Включить / выключить перевод |
| `Ctrl + Shift + L` | Переключить язык перевода по кругу |
| `Ctrl + Shift + C` | Очистить кэш переводов |

## ⚙️ Настройки

### Основные

**Включить перевод**
* Полностью включает или отключает отображение перевода.
* По умолчанию: `Включено`

**Язык перевода**
* Выберите язык, на который будет переводиться текст.
* Доступно 17 языков, включая русский, английский, немецкий, испанский, китайский и другие.
* По умолчанию: `Русский`

### Внешний вид

**Размер текста**
* Регулирует размер шрифта перевода от 8 до 32 пикселей.
* По умолчанию: `18`

**Цвет текста**
* Выберите любой цвет для текста перевода с помощью палитры.
* По умолчанию: `Белый с 80% прозрачностью`

**Шрифт**
* Выберите один из предустановленных шрифтов или укажите свой.
* **Свой шрифт**: Введите название системного шрифта или цепочку шрифтов через запятую (например, `'Segoe UI', Roboto, Arial, sans-serif`). Названия с пробелами обязательно заключайте в кавычки.
* По умолчанию: `По умолчанию`

**Показывать иконку**
* Отображает или скрывает иконку перевода рядом с текстом перевода.
* По умолчанию: `Включено`

**Иконка перевода и её цвет**
* Выбрать вид иконки (например, кошечка, червячок, музыка, блокнот и остальные) и её цвет.
* По умолчанию: иконка «языки» с белым цветом.


## 💡 Советы

* Если перевод перестал работать, попробуйте очистить кэш горячей клавишей `Ctrl + Shift + C`.
*При первом включении обязательно выберите перевод и выберите размер текста*
* Для ввода **своего шрифта** указывайте названия установленных в системе шрифтов. В Windows их можно найти в `Параметры` → `Персонализация` → `Шрифты`.

## 🛠️ Установка

1. Скачайте последнюю версию аддона из каталога PulseSync.
2. Установите его, перетащив файл `.pext` в окно приложения PulseSync.
3. Включите любой трек с синхронным текстом в Яндекс Музыке, и перевод появится автоматически.
`К сожалению, магия перевода с английского на английский или с русского на русский недоступна Translyric.`
